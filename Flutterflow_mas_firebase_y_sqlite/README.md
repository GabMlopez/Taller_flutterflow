# Aplicación de Mixer multiusuario
Esta aplicación permite subir archivos de audio, modificarlas y guardar mixes personalizados online u offline.
## Ingresar a la Aplicación
Al abrir la aplicación aparece una pantalla de inicio de sesión.
Se puede elegir entre 2 opciones:
* Iniciar sesión
* Registrarse con un nuevo usuario
### Iniciar sesión
Se ingresan las credenciales y se validan para permitir el ingreso
### Registrarse
Se pide usuario y contraseña de al menos 6 caracteres.
Una vez registrado debe iniciar sesión con el nuevo usuario.

## Funcionalidades principales

###   Mezclador de Audio
Al ingresar a la aplicación, se accede a la sección del mezclador de audio que permite modificar las pistas de audio cargadas y reporducidas.
El mezclador tiene un control de velocidad que puede reducir hasta la velocidad de reproducción hasta 0.5 de la original o aumentarla al doble
Además, hay una opción para activar un efecto de eco sobre el audio.

### (Samples) Pistas de Audio
Aquí se almacenan las pistas de audio cargadas y modificadas. Las pistas se pueden reproducir para el mezclador.
Las pistas también se pueden marcar como favoritas para poder conservarlas aún sin conexión.

### Chats de proyectos
Estos chats se crean para colaborar en un proyecto con el mezclador. Al ingresar al chat, automáticamente se agrega al usuario a ese chat si es nuevo. 
Los mensajes se actualizan en vivo, y varios usuarios pueden conectarse a la vez.
